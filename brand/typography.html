Primary Typeface

FUTURA PT

Futura PT is the primary typeface for Zeb's Platinum Fitness. Its geometric construction conveys strength and modernity while remaining highly legible. Use Century Gothic or system sans-serif as fallbacks.

Guidelines

USAGE RULES

HEADLINES

  • Always uppercase
  • Bold weight (700)
  • Letter-spacing: 4-8px depending on size
  • High contrast against background
  • Keep headlines short and impactful

BODY TEXT

  • Light weight (300) for elegance
  • Line-height: 1.7-1.8 for readability
  • Gray (#888) on dark backgrounds
  • Regular sentence case
  • Maximum 70 characters per line

LABELS & NAVIGATION

  • Small size (10-11px)
  • Medium weight (500)
  • Uppercase with letter-spacing 2-3px
  • Gold for active/accent states
  • Gray for secondary labels

QUOTES

  • Italic style for distinction
  • Gold color for brand alignment
  • Slightly larger than body (1.125-1.5rem)
  • Use for testimonials and key phrases
  • Attribution in uppercase, small size

Implementation

CSS VARIABLES

:root {
  /* Typography */
  --font-primary: 'Futura PT', 'Century Gothic', 'Futura', sans-serif;
  --font-weight-light: 300;
  --font-weight-normal: 400;
  --font-weight-medium: 500;
  --font-weight-bold: 700;
}

/* Headlines */
h1, h2, h3, h4, h5, h6 {
  font-weight: var(--font-weight-bold);
  text-transform: uppercase;
  letter-spacing: 4px;
  line-height: 1.2;
}

/* Body */
body {
  font-family: var(--font-primary);
  font-weight: var(--font-weight-light);
  line-height: 1.7;
}

/* Labels */
.label {
  font-size: 11px;
  font-weight: var(--font-weight-medium);
  text-transform: uppercase;
  letter-spacing: 3px;
}
