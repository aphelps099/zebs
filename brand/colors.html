Guidelines

COLOR USAGE

DO

  • Use black as primary background
  • Apply gold for accents, CTAs, and highlights
  • Use white for primary body text on dark backgrounds
  • Use gray (#888) for secondary text
  • Maintain high contrast ratios for accessibility
  • Use light gray (#F5F5F5) for alternate section backgrounds

DON'T

  • Use gold as a background color
  • Place gold text on light backgrounds
  • Mix yellow and gold in the same composition
  • Use colors outside the defined palette
  • Add gradients or color effects
  • Reduce contrast below WCAG AA standards

Implementation

CSS VARIABLES

Copy this code block into your stylesheet root for consistent color application.

:root {
  --black: #000000;
  --near-black: #0a0a0a;
  --dark-gray: #111111;
  --mid-gray: #1a1a1a;
  --gray: #888888;
  --light-gray: #F5F5F5;
  --white: #FFFFFF;
  --gold: #ffcc00;
  --gold-light: rgba(255, 204, 0, 0.15);
  --yellow: #fdea01;
}
