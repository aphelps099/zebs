# Zeb's Platinum Fitness — Brand House

The strategic brand hub for Zeb's Platinum Fitness, a personal training studio serving adults 50+ in Redding, CA.

**Live Site:** [zebsplatinumfitness.com](https://www.zebsplatinumfitness.com/)

---

## Overview

This Brand House contains all marketing assets, templates, playbooks, and product information for building Zeb's digital presence and scaling the business online.

### Brand Essence

- **Target Audience:** Adults 50+
- **Tagline:** Structure • Motivation • Accountability
- **Style:** Bold, minimalist, Futura-based typography. Dark mode. Gold accents.

## Design System

### Colors

| Color | Hex | Usage |
|-------|-----|-------|
| Black | `#000000` | Primary background |
| Gold | `#ffcc00` | Accents, CTAs, highlights |
| White | `#FFFFFF` | Primary text |
| Gray | `#888888` | Secondary text |
| Dark Gray | `#111111` | Card backgrounds |
| Light Gray | `#F5F5F5` | Alternate sections |

### Typography

- **Primary Font:** Futura PT (Century Gothic fallback)
- **Headlines:** Bold, uppercase, letter-spacing 4-8px
- **Body:** Light weight (300), line-height 1.7-1.8
- **Labels:** 10-11px, uppercase, letter-spacing 2-3px
